# Replace false layout choices with onboarding and real workspace modes

## User impact

First launch promises three layouts that currently render identically and skips the setup required for a successful first run.

## Acceptance criteria

- Setup is resumable and validates runtime, models, workspace, and safety.
- Focus mode is complete.
- Studio and Orchestrate are advertised only if their distinct acceptance tests pass.
- Readiness is visible after onboarding.

## Dependencies and blockers

NH-0001 and design approval.

## Progress

The six-step onboarding prototype and three run-mode selectors are coded. Focus is functional; Studio and Orchestrate remain clearly labeled Preview.
